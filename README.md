
# Sleact

## 설명

> 메신저 웹 Slack을 구현한 클론 코딩 프로젝트입니다.</br>
> 백엔드가 구축되어 있고 백엔드 개발자로부터 API.md와 typings/db.sts를 전달받았다는 가정하에 프론트 개발만 진행하였습니다.</br>
> React + Typescript 기반으로 개발되었고, 전역 상태관리는 SWR을 이용했습니다.

## 실행 방법

```
//root dir
npm install

//back dir
npm run dev

//alecture dir
npm run dev
```
